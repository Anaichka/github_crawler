# github_crawler
Simple little crawler to retrieve github content using given input information

To start crawler just add to the input.py JSON containing:

Example:
{
  "keywords": [
    "openstack",
    "nova",
    "css"
  ],
  "proxies": [
      "80.48.119.28:8080",
      "110.34.13.4:8080",
      "8.210.83.33:80",
      "103.47.67.158:8080"
      ],
  "type": "Repositories"
}
